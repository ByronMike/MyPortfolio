Projets

Projet n°1 : création d'un site web

Surf-Community.fr

  • Choix technique : WordPress | Elementor Pro
  • Détails techniques: sécurité, optimisation SEO, performance web, responsive, RGPD
  • Visibilité: +500k visiteurs depuis 2017

Surf-Community.fr est un site internet créé en 2017 créé et alimenté par mes soins, en lien avec ma passion pour l’univers Surf. Ce site est dédié à la communauté Surf francophone et vise à donner des conseils sur le Surf : matériel, technique, préparation physique, actualités, etc. Surf-Community.fr est également présent sur les réseaux sociaux, dont Instagram (+13k abonnés).

Projet n°2 : création d'une application mobile

MyQuiver

  • Choix technique : Glide* (NoCode)
  • Détails techniques: application publique, synchronisation en temps réelle
  • État: application opérationnelle (en cours de remplissage)

MyQuiver est une application mobile créée et alimentée par mes soins, qui vise à décrire finement (catégorie, niveau conseillé, note, vidéo, technologie, shape, dimensions, etc.) les planches de surf proposées sur le marché et à permettre à l’utilisateur de créer un Quiver virtuel.


* Glide est un outil no code permettant la création d'une application mobile. L'application GlideApp est une progressive web app (PWA) qui se partage à l'aide d'un lien URL.
